# Renewable_Energy_Analytics
# 📌 Dashboard_Overview
This project analyzes renewable energy generation, consumption, carbon emission reduction, and environmental factors across different states and regions. The dashboard helps stakeholders monitor renewable energy performance, track sustainability goals, and identify trends in energy production.

# 🛠 Tools & Technologies
Power BI Desktop
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel / CSV
Data Modeling (Star Schema)

# 📊 Power BI Visualizations Used
- KPI Cards
- Clustered Bar Charts
- Donut Charts
- Treemaps
- Slicers
- Funnel
- Area Chart

# 📂DataSet
The dataset usedin this project was obtained from Chatgpt and contains information such as energy_id,date,state,region,energy_source,generation,consumption,etc.
Source : Chatgpt

# 🔍 Key Features
✅ Interactive dashboard with dynamic filters and slicers.
✅ State-wise and region-wise renewable energy performance analysis.
✅ Top 10 states by energy generation visualization.
✅ Best and lowest performing state identification.
✅ Energy source contribution analysis (Solar, Wind).
✅ Temperature and wind speed correlation with energy generation.
✅ Time-series analysis of energy generation and consumption trends.
